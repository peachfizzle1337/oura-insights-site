# Oura Insights

Oura Insights is a personal analytics project that connects to the Oura API, stores authorized health data locally, and generates weekly and monthly summaries with simple tag-based pattern analysis.

This project is intended for personal use and focuses on:

- weekly health summaries
- monthly trend summaries
- observational comparisons between Oura metrics and user-created tags

The application is not a medical device and does not provide medical advice.
