# Privacy Policy

Oura Insights is a personal-use application for accessing and analyzing Oura data that the user explicitly authorizes through the Oura API.

## Data Use

The application uses authorized Oura data only to generate personal summaries, trend reports, and observational insights.

## Data Storage

By default, data retrieved by the application is stored locally by the user. The project is designed for personal analytics and does not intentionally sell or share data with third parties.

## Access Revocation

The user may revoke Oura API access at any time through their Oura account or developer application settings.

## No Medical Advice

Any summaries or patterns generated by the application are informational only and are not medical advice.
